# Triviape Roadmap

## Q3 2025: Core Experience Refinement

*   **Daily Quiz Enhancements:**
    *   [ ] Introduce new question types (e.g., image-based, fill-in-the-blank).
    *   [ ] Add more variety to the daily quiz categories.
    *   [ ] Implement a system for users to rate questions and provide feedback.
*   **Leaderboard Improvements:**
    *   [ ] Add weekly and monthly leaderboards.
    *   [ ] Introduce friend-specific leaderboards.
    *   [ ] Add more stats and insights to the leaderboard page.
*   **Performance Optimization:**
    *   [ ] Continue to optimize Core Web Vitals.
    *   [ ] Improve the performance of the multiplayer lobby.

## Q4 2025: Social and Community Features

*   **Friend Challenges:**
    *   [ ] Implement the ability for users to challenge their friends to a quiz.
    *   [ ] Add a notification system for challenges.
*   **Teams/Clans:**
    *   [ ] Introduce the concept of teams or clans that can compete against each other.
    *   [ ] Add team-based leaderboards.
*   **Public Profiles:**
    *   [ ] Enhance public profiles with more stats, achievements, and customization options.

## Q1 2026: Monetization and Growth

*   **Premium Subscription:**
    *   [ ] Introduce a premium subscription with features like an ad-free experience, exclusive quizzes, and advanced stats.
*   **In-App Purchases:**
    *   [ ] Add the ability to purchase cosmetic items (e.g., profile themes, avatars) with coins.
*   **Referral Program:**
    *   [ ] Implement a referral program to encourage user growth.

## Future Ideas (Backlog)

*   **User-Generated Quizzes:** Allow users to create and share their own quizzes.
*   **Live Tournaments:** Host scheduled live tournaments with prizes.
*   **Deeper AI Integration:** Use AI to generate personalized quizzes and learning paths.
*   **Mobile App:** Develop a native mobile app for iOS and Android.
